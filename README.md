## Description

Durgapuja Countdown is a beautiful animated countdown web app that shows the remaining time until Durga Puja Sasthi.
It features:
- Glassmorphism UI
- Smooth rolling digit animation
- Responsive design
- Bengali language interface
- Perfect for festive websites and personal projects.

## Features
- 🎯 Real-time countdown
- 🎞️ Smooth roller animation digits
- 📱 Fully responsive design
- 🌙 Modern glass effect UI
- 🇮🇳 Bengali language support
- ♿ Reduced motion accessibility support

## Installation

Clone the repository:
````Bash 

git clone https://github.com/codegear-2011/Durgapuja-countdown.git
````
Go to the project folder:
````Bash
cd Durgapuja-countdown
````
Then simply open:
````

index.html
````
No dependencies required. Pure HTML, CSS & JavaScript.
---
## Usage
You can modify the countdown target date inside the script:
`````JavaScript
const targetDate = new Date("October 17, 2026 00:00:00").getTime();
`````
Change the date according to the next Durga Puja Sasthi.
